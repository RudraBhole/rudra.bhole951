Tips

Don't use AI

AI might generate code for you, but if you don’t write it yourself, you won’t truly understand how it works.

Organize

Especially for big websites, keeping your files and folders tidy helps you find code easily and avoid confusion.


Practice Regularly

Coding is like a skill; if you don’t practice, you may forget how to solve problems or write code efficiently, as I experienced myself.
