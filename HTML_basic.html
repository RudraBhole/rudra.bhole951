HTML Basic

In learning HTML basics, you’ll discover the essential tags that form the foundation of every webpage. These tags are like the visible “building blocks” or physical evidence of the site — they tell the browser what to show, such as headings, paragraphs, images, links, and other content. Understanding these tags helps you structure your page clearly so visitors can easily read and navigate it.




<h1>


We begin with a start tag: <h1>
Then some text: "This is a heading."
Then a matching end tag: </h1>
Content needs to go in between tags, but content and tags don't need to be on the same line.

<h1>
I look the same on one line!
</h1>

<h1>This is a level 1 heading</h1>
<h2>This is a level 2 heading</h2>
<h3>This is a level 3 heading</h3>
<h4>This is a level 4 heading</h4>
<h5>This is a level 5 heading</h5>
<h6>This is a level 6 heading</h6>

<p>


<p>This is a paragraph</p>

The <p> tag defines a paragraph in HTML.

It is used to group and display blocks of text on a webpage.
Paragraphs help organize content and make it easier for users to read.
Browsers automatically add space before and after a paragraph to separate it from other elements.

<em>


The <em> element (short for emphasis) makes text lean to the right.

Emphasis is used to make words stand out. It indicates that the text is important, or that it should be read in a different voice.

Example:
<p><em>Alice</em> didn't say "bananas".</p>

<strong>


Another element used for emphasis is <strong>.

It's used to indicate some words are important — even more important than <em>. It can also mean the words are serious or urgent.

Example:
<p> You should make sure to <strong>arrive by 9:30 AM</strong>. </p>

<br>


Browsers do ignore extra spaces and new lines in code. How do you start a new line without starting a new paragraph, then?

You can use the <br> (line break) element to start a new line.

Example:
<p>
  Consider me <br>
  as one who loved poetry
  and persimmons.
</p>

Unlike other elements, the <br> element doesn't have a closing tag. The </br> tag doesn't exist.

Discription of the code and how it works

The original code is a simple HTML content structure that presents information about Sport Climbing.

<h1> is used for the main page heading: Sport Climbing.

<h2> tags create section headings like Origins, Indoor Climbing, and Olympic Recognition.

<p> tags hold paragraphs of descriptive text for each section, explaining what Sport Climbing is, its indoor popularity, and its Olympic debut.

When run in a browser, it will display a large main title, followed by smaller section headings and readable paragraphs beneath each heading.

<h1>Sport Climbing</h1>

<h2>Origins</h2>

<p>Sport Climbing is a form of rock climbing that relies on permanent anchors fixed to the rock for protection, in
which a rope that is attached to the climber is clipped into the anchors to arrest a fall, or that involves climbing
short distances with a crash pad underneath as protection. Sport Climbing emphasises strength, endurance, gymnastic
ability and technique.</p>

<h2>Indoor Climbing</h2>

<p>With increased accessibility to climbing walls, and gyms, more climbers now enter the sport through indoor climbing
than outdoor climbing.</p>

<h2>Olympic Recognition</h2>

<p>Sport Climbing was scheduled to make its Olympic debut at the 2020 Summer Olympics in Tokyo, Japan, and was
previously tested at the 2018 Summer Youth Olympics.</p>
