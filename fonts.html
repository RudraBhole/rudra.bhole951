Fonts

CSS structure helps you control how the webpage looks by letting you select HTML elements and apply different styles to them. You'll learn about selectors that target specific parts of the page, and how to use properties to change colors, fonts, sizes, spacing, and layout. This gives you the power to make your website visually appealing and easy to use, turning plain content into a polished design.

Google fonts are a type of web font. These are downloaded when a page is loaded, so they work even if your computer doesn't have them installed!





Paste the @import into the first line of your css

font-family{}


A font family is a group of fonts that share a specific style. There are many different font families to choose from!

We can change the font family of an element using the font-family property:

@import url('https://fonts.googleapis.com/css2?family=Seaweed+Script&display=swap');

@import url('https://fonts.googleapis.com/css2?family=Offside&display=swap');

h1 {
font-family: Seaweed Script;
(and other css)
}

h2 {
font-family: Offside;
(and other css)
}

How it looks all together

What is this code and why doesn’t it show anything on the page?

This webpage uses custom Google Fonts to make the text look more attractive and professional.

The main heading, "Sweet Crumbs Bakery," is styled with the Great Vibes font, which is a decorative, script-style font that gives the bakery name a charming and inviting personality.

All other text, including the subheading and paragraphs, uses the Merriweather font, a clean and readable serif font that makes the content easy on the eyes.

By importing these fonts from Google Fonts and applying them via CSS, the page looks much more visually appealing than using default browser fonts.

Centering the text with text-align: center also gives the page a neat and organized appearance, making it simple and elegant for visitors to read.

Overall, using fonts in this way helps convey the bakery’s style and enhances the user experience.

<!DOCTYPE html>
<html>
<head>
<style>
@import url('https://fonts.googleapis.com/css2?family=Great+Vibes&display=swap');

@import url('https://fonts.googleapis.com/css2?family=Merriweather&display=swap');

body {
font-family: 'Merriweather', serif;
text-align: center;
}

h1 {
font-family: 'Great Vibes', cursive;
}

</style>
</head>

<body>

<h1>Sweet Crumbs Bakery</h1>

<h2>Freshly Baked Goodies Every Day</h2>

<p>At Sweet Crumbs, we believe every bite should feel like home.</p>

<p>Come visit us for your daily bread, a sweet treat, or a cup of coffee.</p>

</body>

</html>
