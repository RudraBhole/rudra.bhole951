CSS Structure

CSS structure helps you control how the webpage looks by letting you select HTML elements and apply different styles to them. You’ll learn about selectors that target specific parts of the page, and how to use properties to change colors, fonts, sizes, spacing, and layout. This gives you the power to make your website visually appealing and easy to use, turning plain content into a polished design.

How to add a CSS file?


Visual Studio Code:

1. Click add file
2. Name the file "main.css"
3. Start coding in CSS

Playground:

Use the tag
<style>

</style>
to code in CSS. Inside the tag add your CSS.

<link>


Note: This example is for Visual Studio Code.

A CSS file needs to be linked to a HTML file with a <link> element.

To link our main.css file, we add a <link> element to our index.html file inside the head tag, like this:

<link rel="stylesheet" href="main.css">

We set two attributes on <link>:

- rel: short for relationship. Here we say we're linking to a stylesheet.
- href: the location of the stylesheet, which is the file name.

Now the styles we define in our CSS file will be applied to our HTML document.

What is CSS?


CSS (Cascading Style Sheets) is the language that adds style and personality to your website.

While HTML creates the structure of a page (like headings, paragraphs, and images), CSS decides how those elements should look.
With CSS, you can change colors, fonts, and sizes, add spacing between elements, arrange content into columns, and even create animations.

The word “cascading” means that styles follow a set of rules about which ones take priority if multiple styles apply to the same element.
This gives you flexibility and control in designing your webpage.

In simple words: HTML is the skeleton, CSS is the skin and clothing — it makes your website visually appealing, easy to read, and professional.

Elements


In CSS, you write rules in a simple pattern.

1. First, choose an element (like p, h1, or div).
2. After it, add curly brackets { }.
3. Inside the brackets, write a property and a value.
4. End with a semicolon ;

Example:

h1 {
color: red;
text-align: center;
}

This means all h1 headings will be red and in the center of the page.
