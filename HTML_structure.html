HTML Structure

The HTML structure is the backbone of any web page. It defines how content is organized using tags. A well-structured HTML document helps browsers understand the layout, improves accessibility for all users, and makes it easier for developers to maintain and update the site. It's like building a strong foundation before decorating a house.




How to Setup a File


Steps:

1. Open VS Code
2. Click "File" > "Open Folder"
3. Create a new folder and name it something
4. Click add file
5. Name the file "index.html"
6. Start coding in html

What is HTML?


    HTML stands for Hyper Text Markup Language
    HTML is the standard markup language for creating Web pages
    HTML describes the structure of a Web page
    HTML consists of a series of elements
    HTML elements tell the browser how to display the content
    HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Elements


What is an HTML Element?
An HTML element is defined by a start tag, some content, and an end tag:

<tagname> Content goes here... </tagname>

The HTML element is everything from the start tag to the end tag:
<h1>My First Heading</h1>
 <p>My first paragraph.</p>

<html>


What is <html>?

The <html> tag is the root of every HTML document. It wraps all the content you see on a web page—like text, images, and links. Think of it as the container that holds the entire webpage structure. Everything inside the <html> tag tells the browser how to display the page.

How to use it? <html>?

1. Start your HTML file with <html>.
2. Write all your HTML content between <html> and </html>.
3. The <html> tag wraps the entire webpage structure.
4. Make sure you close it with </html> at the very end of your file.

<head>


The <head> tag is the part of an HTML page that holds information about the page—like its title, styles, and other settings.

It can also link to stylesheets that control how the page looks (you’ll learn about stylesheets later).

This information doesn’t show up on the page itself, but it helps the browser know how to display it.

<body>


The <body> tag contains all the content that you see on the webpage, like text, images, links, and videos.
Everything inside the <body> tag is visible to visitors when they open the page in a browser.

It is the main part of the HTML document where you add everything you want people to see and interact with.

This includes headings, paragraphs, buttons, forms, and any other elements that make up the page’s content.

Without the <body> tag, the page would be blank.

How to see your result?


With Live Server

1. Open your HTML project folder in VS Code.
2. Go to the Extensions panel (the square icon on the left).
3. Search for and install the Live Server extension by Ritwick Dey.
4. Once installed, open your main HTML file (e.g., index.html).
5. Right-click anywhere in the editor and select “Open with Live Server”. 6. Your default browser will open and display your webpage live.
7. Now, whenever you SAVE (ctrl/cmd+s) changes to your HTML or CSS files, the browser will automatically refresh to show the updates.

Playground

1. Go to the playground, the button is located on the top left corner or in the sidebar of every webpage.
2. Type your code in the first textbox.
3. Click "Run Code" and see you your website below.

This is a great way to see your website as you build it, making development faster and easier!
